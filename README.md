# istighfar-site

Public-facing pages for Istighfar &amp; Dhikr Counter:

- `index.html` — landing page
- `support.html` — support / FAQ (linked from the App Store as Support URL)
- `privacy.html` — privacy policy (linked from App Store Connect as Privacy Policy URL)
- `style.css` — shared minimal stylesheet

Hosted on GitHub Pages. The app itself lives in a separate private repo.
